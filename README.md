# 📊 Excel Sales Analysis Project
## 🔹 Project Overview
This project demonstrates my skills in Excel Data Analysis using Pivot Tables, Data Modeling, and DAX.
## 🔹 Dataset Used
* Sales Table (Order ID, Product ID, Sales, Quantity, etc.)
* Product Table (Product ID, Product Name, Category)
## 🔹 Key Steps
* Created a relationship between Sales and Product tables using Product ID
* Built a Data Model in Excel (Power Pivot)
* Created DAX measure to calculate Average Sales
* Analyzed Category-wise Average Sales using Pivot Table
## 🔹 DAX Formula Used
DAX_Avg Sales = AVERAGE(sum of sales[sum of qty sold])
## 🔹 Tools & Skills
* Microsoft Excel
* Pivot Tables
* Power Pivot
* DAX
* Data Modeling
## 🔹 Key Insight
Category-wise analysis helps identify which product categories generate higher average sales.
